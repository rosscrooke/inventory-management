System Architecture

Frontend
App.vue — Shell, nav, modals
Views — Dashboard, Inventory, Orders, Demand, Spending, Reports, Backlog
Components — FilterBar, Modals, ProfileMenu, Charts
Composables — useFilters, useAuth, useI18n
api.js — Axios HTTP client
HTTP/JSON
Backend
main.py — FastAPI app, routes, CORS
Pydantic Models — InventoryItem, Order, DemandForecast, BacklogItem, Transaction
Filter Logic — warehouse, category, status, month/quarter
Computed Aggregates — dashboard summary, quarterly reports, spending
Load once
Data Layer
mock_data.py — Reads JSON on startup, exposes in-memory lists
server/data/ — 6 JSON files, loaded once, never written to
No persistence — Restart reloads original data

Request Data Flow

1

User Changes Filter

FilterBar.vue updates singleton ref in useFilters composable

2

Watcher Fires

View component watches filter refs, calls loadData() on change

3

API Call

api.js builds URLSearchParams, skips params with value 'all', calls FastAPI

4

Backend Filters

FastAPI applies warehouse → category → status → month filters sequentially on in-memory lists

5

Reactive Update

Response stored in ref, computed properties recalculate, Vue re-renders

API Endpoints

GET /api/inventory warehouse, category filters
GET /api/inventory/{id} single item by ID
GET /api/orders warehouse, category, status, month filters
GET /api/orders/{id} single order by ID
GET /api/dashboard/summary all 4 filters, computed KPIs
GET /api/demand no filters
GET /api/backlog no filters, adds has_purchase_order flag
GET /api/spending/summary totals + % changes
GET /api/spending/monthly monthly breakdown by cost type
GET /api/spending/transactions recent transaction list
GET /api/reports/quarterly aggregated quarterly stats
GET /api/reports/monthly-trends month-over-month order & revenue

Frontend State — Composables

useFilters.js
Global singleton filter state. All views share the same refs so the FilterBar controls every page simultaneously.
selectedPeriod selectedLocation selectedCategory selectedStatus getCurrentFilters() resetFilters() hasActiveFilters
useAuth.js
Mock user session. Exposes locale-aware user profile and task list. No real auth — demo only.
currentUser getInitials() logout()
useI18n.js
Internationalisation. Supports English and Japanese. Locale persisted to localStorage.
t(key) currentLocale setLocale() currentCurrency translateProductName() translateWarehouse()
